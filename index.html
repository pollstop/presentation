A simple poll application

  • Create a poll (with choices and relevant tags)
  • Send link to friends, share it
  • People will vote, see results instantly
  • End poll whenever you want

Extra Functionality

  • Ability to assign start and end dates
  • Ability to change your decision later

KISS UI

Allow users to naturally react to application without needing documentation.

 

Quickly view ongoing polls,

Vote them instantly.

 

No complex poll options before creating them.
