Powerful Technologies

  • Powerful REST API (django)
  • PostgreSQL for scalability
  • Progressive web app with React & ES6
  • One-click login using Google
  • Continuous integration with TravisCI

Minimal User Interface

  • Responsive design for best experience
  • Single click voting
  • Results remains hidden until
    - poll is ended
    - user has voted
  • Easy sharing using share buttons for
    - facebook
    - twitter
    - etc.

A simple poll application

  • Create a poll (with choices and relevant tags)
  • Send link to friends, share it
  • People will vote, see results instantly
  • End poll whenever you want

FB/Twitter

  • In context, not mainly about polls, it is difficult to return back to a poll afterward
  • Unable to search polls by category
  • Results displayed in weird ways.
  • Vendor lock-in (you need to sign-in etc)

Pollstop

  • Only polls; questions and their answers in a simple user interface
  • Filter polls by categories
  • You can see results easily
  • No need to signup
  • Data-mining friendly
